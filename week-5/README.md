# Week 5 - React Hands-On

This week covers React fundamentals through hands-on labs.

## Topics Covered
- Functional and Class Components
- JSX syntax
- Props and State management
- Lifecycle methods (componentDidMount, ngOnInit)
- React Router for navigation
- Fetching data from APIs
- Lists, conditional rendering, and array methods in JSX

## Hands-On Labs

| File | What it covers |
|------|----------------|
| App.jsx | Basic React app with JSX |
| MultiComponent.jsx | Multiple functional components |
| CalculateScore.jsx | JSX expressions and computed values |
| BlogPosts.jsx | Class component + API call with componentDidMount |
| CohortDetails.jsx | Array rendering with conditional styling |
| TrainerRouter.jsx | React Router with BrowserRouter, Routes, useParams |
| ShoppingCart.jsx | Passing props to child components |
| MallCounter.jsx | Class component with state and button events |
| CricketApp.jsx | Array filter, spread, conditional rendering |
| OfficeRental.jsx | Object and array rendering with inline styles |
