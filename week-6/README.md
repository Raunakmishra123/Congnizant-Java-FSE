# Week 6 - Angular and Git

This week covers Angular framework fundamentals and Git version control workflows.

## Topics Covered

### Angular
- Angular project structure and standalone components
- Component lifecycle hooks (ngOnInit, ngOnDestroy, ngOnChanges)
- Input/Output decorators and EventEmitter
- Custom directives and pipes
- Template syntax (ngIf, ngFor, trackBy)
- Component communication patterns

### Git
- Repository setup and configuration
- Staging, committing, and pushing
- .gitignore usage
- Branching and merging
- Handling merge conflicts
- Remote repository workflows
