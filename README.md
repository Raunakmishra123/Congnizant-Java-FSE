# Cognizant Digital Nurture 5.0 - Complete Java DSA Training

Welcome to the comprehensive Cognizant DSA (Data Structures & Algorithms) and Java training program. This repository contains production-quality implementations of core Java concepts taught at Cognizant interview level.

## 📚 Curriculum Overview

### Week 1 Content

#### 1. Design Patterns (01-DesignPatterns)
- **Singleton Pattern**: Thread-safe singleton implementations (Eager, Lazy, Bill Pugh)
- **Factory Pattern**: Object creation without specifying exact classes
- **Builder Pattern**: Constructing complex objects step by step
- **Observer Pattern**: One-to-many dependency implementation
- **Strategy Pattern**: Encapsulating different algorithms

#### 2. Data Structures & Algorithms (02-DataStructures)
- **Big O Analysis**: Time & Space Complexity classes
- **Linked Lists**: Singly Linked List operations
- **Searching**: Linear and Binary Search comparison
- **Sorting**: Bubble, Quick, Merge, and Insertion Sort
- **Recursion**: Tail recursion, Fibonacci, Factorial, and String Reverse

#### 3. Test Driven Development (04-TestDrivenDevelopment)
- JUnit 5 fundamentals and assertions (assertEquals, assertThrows, etc.)
- AAA Pattern (Arrange, Act, Assert)

#### 4. Logging Framework (05-Logging)
- SLF4J (Simple Logging Facade for Java) and log levels (TRACE, DEBUG, INFO, WARN, ERROR)

### Week 2 Content
- **Spring Core & Maven**: Dependency Injection, Inversion of Control, Beans & Bean Scope, XML Configuration
- **Spring Data JPA & Hibernate**: Object Relational Mapping, JPA annotations, CRUD operations

### Week 3 Content
- **Spring REST & Spring Boot 3**: REST API development, HTTP methods, controllers, Spring Boot starter projects

### Week 4 Content
- **Code Quality & SonarQube**: Clean code principles, static analysis, metrics

### Week 5 Content
- **React Hands-On**: JSX, functional and class components, props, state, lifecycle methods, React Router, API calls, conditional rendering

### Week 6 Content
- **Angular**: Component lifecycle hooks, Input/Output, custom directives and pipes, template syntax
- **Git**: Repository setup, branching, merging, conflict resolution, remote workflows

## 🛠️ Setup Instructions

### Prerequisites
- Java 11 or higher
- Maven 3.6 or higher
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Raunakmishra123/Congnizant-Java-FSE.git
cd Congnizant-Java-FSE
```

2. Build the project:
```bash
mvn clean install
```

3. Run tests:
```bash
mvn test
```
