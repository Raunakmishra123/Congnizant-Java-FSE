# Week 2 - Spring Framework

This week covers the Spring Framework core concepts and Spring Data JPA.

## Topics

### 01 - Spring Core and Maven
- IoC (Inversion of Control)
- Dependency Injection (Constructor and Setter)
- Spring Beans and ApplicationContext
- XML-based configuration
- Maven build tool

### 02 - Spring Data JPA and Hibernate
- ORM (Object Relational Mapping)
- JPA entities and annotations
- Hibernate as JPA implementation
- Basic CRUD with EntityManager

## Hands-on Projects

- **LibraryManagement** - Spring XML config + BookService + Book entity
- **ORM-Learn** - JPA entity mapping with Country class
