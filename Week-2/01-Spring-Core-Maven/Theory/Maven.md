# Apache Maven

Apache Maven is a popular build automation tool used primarily for Java projects. It uses a XML file called `pom.xml` (Project Object Model) to configure dependencies, build settings, and plugins.

## Key Features

1. **Dependency Management:** Automatically downloads libraries and their transitively required dependencies from Central Maven Repository.
2. **Build Lifecycle:** Standardized lifecycle phases:
   - `clean`: Removes files generated by the previous build.
   - `compile`: Compiles the source code.
   - `test`: Runs unit tests.
   - `package`: Packs compiled code into distributable format (JAR/WAR).
   - `install`: Installs package into local repository for use as dependency in other local projects.

## Example pom.xml structure

```xml
<project>
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.example</groupId>
    <artifactId>demo-app</artifactId>
    <version>1.0.0</version>
    
    <dependencies>
        <!-- Dependency declared here -->
    </dependencies>
</project>
```
